# Final QC Checklist

## Visual Continuity
- [ ] Main character faces stay consistent
- [ ] Costume and hair stay consistent
- [ ] Recurring locations still look like the same place
- [ ] Color palette matches the visual bible
- [ ] Key props stay consistent

## Shot Continuity
- [ ] Eyelines match across cuts
- [ ] Shot direction is coherent
- [ ] Inserts and reaction shots support the scene
- [ ] No missing establishing shot where needed

## Sound
- [ ] Ambience is present in every scene
- [ ] Foley supports key actions
- [ ] Dialogue level is clear
- [ ] Music does not overpower emotion

## Delivery
- [ ] Final sequence order is correct
- [ ] Titles / subtitles are checked
- [ ] Export settings are recorded
